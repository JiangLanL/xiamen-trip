端午人多备选方案

  • 6/20 植物园太挤：改走“铁路文化公园 → 鸿山公园/鸿山寺 → 钟鼓索道”。交通：如家打车到铁路文化公园约10分钟，铁路文化公园到鸿山公园步行/短打车，鸿山公园到钟鼓索道打车约10分钟。
  • 6/20 沙坡尾艺术西区太挤：改走“沙坡尾避风坞外线 → 大学路 → 民族路”。交通：厦大/南普陀打车到沙坡尾，后面全程步行。
  • 6/20 黄厝沙滩太挤：改去“珍珠湾海边/音乐广场海边短停”。交通：演武大桥/沙坡尾打车到珍珠湾约10-15分钟，珍珠湾回中山路打车约20-30分钟。
  • 6/21 鼓浪屿核心点太挤：最美转角只快速拍照，改走“安海路 → 笔山路 → 鼓新路 → 复兴路 → 港仔后沙滩”。交通：全程步行，少去龙头路中心区久待。
  • 6/21 菽庄花园排队太久：改“港仔后沙滩 → 皓月园外线 → 海边咖啡 → 回安屿1920”。交通：步行为主，必要时坐岛上电瓶车。
  • 6/22 十里长堤太挤：缩短停留，改“龙舟池 → 嘉庚建筑群外观 → 石鼓路 → 全季机场店”。交通：集美内部步行，回全季打车。

必须提前买/约

  • 鼓浪屿船票：6/21 去，6/22 回；东渡/邮轮中心码头 ⇄ 三丘田码头。
  • 钟鼓索道：6/20 10:30 左右场次，分时段票，提前买。
  • 南普陀：免费预约，建议提前约。
  • 厦门大学：想入校必须预约；约不上就走校门外线，不影响后续路线。
  • 植物园、菽庄花园可提前买；日光岩不去，不买。
